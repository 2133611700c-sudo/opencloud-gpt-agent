# OpenClaw Report Policy

All markdown reports produced by OpenClaw task execution must include these required fields:
- `run_id`
- `run_url`
- `task_id`
- `task_type`
- `status`
- `raw_evidence`
- `blocking_issues`
- `next_action`

## Windows readiness report template
Use `windows-readiness-template.md` in this folder for Windows-focused readiness and remediation reporting.
